# Intelligent Travel Planning Agent

## Overview
An intelligent mobile automation agent that plans trips by coordinating across
Google Maps, hotel booking platforms, and flight search tools.

## Features
- Opens Google Maps to explore top places
- Searches hotels on Booking platforms
- Generates flight search links
- Saves everything into a single Google Keep note
- No unsafe actions like payments

## Tech Stack
- Python
- adbutils
- Android (real device)
